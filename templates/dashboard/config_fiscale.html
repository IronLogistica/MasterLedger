🧪 Ambiente di Test SdI COLLAUDO
Prepara con un click tutto il necessario per collaudare l'XML FatturaPA con il software di controllo dell'Agenzia delle Entrate: un cliente finto (CLIENTE TEST SRL, P.IVA 99999999990 con checksum valido, SDI 0000000), un prodotto finto (SEGN-TEST, giacenza 999) e — solo se i campi sono ancora vuoti — un cedente di prova (P.IVA 12345678903). I dati reali già inseriti NON vengono toccati.
Percorso di collaudo (senza toccare il WMS): Ciclo Cliente → Fattura cliente → CLIENTE TEST SRL, riga "Cartello di prova SEGN-TEST", imponibile 250, IVA 22 → registra → scarica XML dal dettaglio → carica il file su fatturapa.gov.it → Controlla la Fattura. Il file di collaudo non va mai firmato né trasmesso allo SdI.
