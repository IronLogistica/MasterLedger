# Caricamento con GitHub Desktop

1. Estrai lo ZIP in una cartella del computer.
2. Apri GitHub Desktop e scegli **File → Add local repository**.
3. Seleziona la cartella `MasterLedger`. Se richiesto, scegli **Create a repository**.
4. Scrivi un messaggio come `Prima versione MasterLedger`, poi fai **Commit to main**.
5. Seleziona **Publish repository** e scegli se tenerlo privato o pubblico.

Prima di pubblicare, crea un file `.env` locale partendo da `.env.example` e inserisci una chiave segreta reale. Il file `.env` è escluso dal caricamento.
